# 🌍 Wanderlust

A full-stack travel listing web application built with Node.js, Express, MongoDB, and EJS. Users can create, edit, delete, and explore travel listings with secure authentication and image uploads.

## 🚀 Features

- User Registration & Login
- Authentication with Passport.js
- Create, Edit & Delete Listings
- Image Upload using Cloudinary
- Reviews and Ratings
- Flash Messages
- Session Management
- MongoDB Atlas Database
- Responsive UI

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- EJS
- Passport.js
- Cloudinary
- Multer
- Bootstrap

## 📂 Installation

Clone the repository

```bash
git clone https://github.com/sarita2233/wanderlust.git
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm start
```

Visit

```
http://localhost:8080
```
